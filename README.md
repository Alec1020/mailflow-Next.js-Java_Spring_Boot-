# AI Mail Manager (Next.js + Spring Boot)

An AI-powered mail and message management system built with **Next.js (frontend)** and **Spring Boot (backend)**.  
The system allows users to send, receive, organize, and intelligently process emails/messages using AI features such as summarization, categorization, and smart replies.

---

## 🚀 Features

### 📩 Core Mail System
- Send and receive messages/emails
- Inbox, Sent, Drafts management
- Read/unread status tracking
- Search and filter messages

### 🤖 AI Features
- AI email summarization
- Smart reply suggestions
- Spam detection
- Message categorization (Work, Personal, Important, etc.)
- Auto-tagging messages

### 🧑‍💻 User Features
- User authentication (JWT-based)
- Profile management
- Message history tracking
